## Descripción del Proyecto

UniStay es una plataforma web diseñada para conectar a estudiantes universitarios que buscan alojamiento cercano a sus campus con propietarios que ofrecen habitaciones en alquiler. Su objetivo principal es facilitar la búsqueda de espacios seguros, confiables y accesibles para aquellos estudiantes que viven lejos y necesitan un lugar donde hospedarse durante su formación académica.
